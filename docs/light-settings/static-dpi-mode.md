> brigtness is a slider with 8 steps

## Set Static DPI mode Option with initial state with speed 8

Request

```txt
0000   1c 00 a0 09 0f 0a 88 a3 ff ff 00 00 00 00 1b 00   ................
0010   00 03 00 05 00 00 02 15 00 00 00 00 21 09 05 03   ............!...
0020   02 00 0d 00 05 0f 01 50 03 a8 00 ff 00 01 04 01   .......P........
0030   ff                                                .
```

Reponse:

```txt
0000   1c 00 a0 09 0f 0a 88 a3 ff ff 00 00 00 00 08 00   ................
0010   01 03 00 05 00 00 02 00 00 00 00 03               ............
```
