## Configuring Breathing Option, with initial state 

> speed is a slider with 5 steps
> The color field allows you to pass a color in RGB.

## Set Breathing Option, with initial state with speed 3 (R: 0 G: 255 B: 0)

Request

```txt
0000   1c 00 a0 e9 80 0b 88 a3 ff ff 00 00 00 00 1b 00   ................
0010   00 03 00 05 00 00 02 15 00 00 00 00 21 09 05 03   ............!...
0020   02 00 0d 00 05 0f 01 20 03 a8 00 ff 00 01 04 01   ....... ........
0030   cf                                                .
```

Reponse:

```txt
0000   1c 00 a0 e9 80 0b 88 a3 ff ff 00 00 00 00 08 00   ................
0010   01 03 00 05 00 00 02 00 00 00 00 03               ............
```
