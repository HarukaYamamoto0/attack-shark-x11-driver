## Configuring Breathing DPI, with initial state 

> speed is a slider with 5 steps

## Set Breathing DPI, with initial state with speed 3

Request

```txt
0000   1c 00 a0 d9 0e 0a 88 a3 ff ff 00 00 00 00 1b 00   ................
0010   00 03 00 05 00 00 02 15 00 00 00 00 21 09 05 03   ............!...
0020   02 00 0d 00 05 0f 01 60 03 a8 00 ff 00 01 04 02   .......`........
0030   0f                                                .
```

Reponse:

```txt
0000   1c 00 a0 d9 0e 0a 88 a3 ff ff 00 00 00 00 08 00   ................
0010   01 03 00 05 00 00 02 00 00 00 00 03               ............
```
